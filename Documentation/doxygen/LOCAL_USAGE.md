Local Doxygen Documentation
===========================

Install Doxygen
---------------

Fedora:

```bash
sudo dnf install doxygen
```

Ubuntu/Debian:

```bash
sudo apt install doxygen
```

Generate Documentation
----------------------

```bash
doxygen ./Doxyfile
```

Use your browser to view the documentation. For example:

```bash
firefox ./html/index.html
```
